# MarshalOptionsEnum
Specifies which records should be returned to the server.  
  
|Constant|Value|Description|  
|--------------|-----------|-----------------|  
|**adMarshalAll**|0|Default. Returns all rows to the server.|  
|**adMarshalModifiedOnly**|1|Returns only modified rows to the server.|
