# DimensionCount Property (ADO MD)
Indicates the number of dimensions on an [axis](./axis-object-ado-md.md).  
  
## Return Values  
 Returns a **Long** integer, and is read-only.  
  
## Applies To  
 [Axis Object (ADO MD)](./axis-object-ado-md.md)
