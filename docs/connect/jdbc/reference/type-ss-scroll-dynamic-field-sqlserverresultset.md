[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

## Syntax  
  
```  
  
public static final int TYPE_SS_SCROLL_DYNAMIC  
```  
  
## Field Value  
 An **int** value of 1006.
